xml (특수 목적 마크업 언어)
W3C에서 개발된 언어로 다른 특수한 목적을 가지는 마크업 언어를
만드는데 사용하도록 권장하는 다목적 마크업 언어.
사용자 임의로 태그를 생성 가능.
웹상에서 정보 교환 가능, 계층적 데이터 구조.

markup language : 문서나 데이터의 구조를 명기하는 언어 중 하나
데이터를 기술하는 정도로만 사용되기에 프로그램 언어와 구별됨.

tag (태그) : 어떤 정보에 메타데이터로 부여된 키워드 또는 분류

metadata (메타데이터) : 어떠한 목적을 가지고 만들어진 데이터

json
속성 - 값 쌍 or 키 - 값 쌍 으로 이루어진 데이터 오브젝트를 전달하기 위해
인간이 읽을 수 있는 텍스트를 사용하는 개방형 표준 포멧.

데이터의 무결성
데이터의 정확성과 일관성을 유지하고 보증하는 것을 가리킨다.
